## Google Analytics

Set your google analytics key in the `config.js` file.
(See [[docs/config|this chapter]] about basic user configurations.)

```js
// config.js
// Google analytics key e.g. G-XXXX
{
	analytics: 'your-google-analytics-key',
}
```
